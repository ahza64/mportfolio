SF secrets

Find out the secrets in San Francisco by either clicking on the map icon, or the tag on the list. Log in, and create your own secrets. Built this from the ground up in 8 days. I utilized Angular abundantly in this app, and made all of my external API calls on the client side to minimize IP pings.

Vagabond

Find out what others think of three different cities by logging in and adding a post, or commenting on others posts. Built in Ruby and RoR, this group project was built in 4 days. I was responsible for project management, setting up a few controllers, database associations and debugging.

@-some-point

Figure out the shortest route to take among the destinations you need to travel. Our first team project. I was responsible for project management, building the controllers and most of the Sql database, setting up Auth, rendering the map, and making the external API call.

proBuilder

Track your projects, the members on those projects, and their assigned tasks. This is my first full stack project. Using MEN technology, I built this app from the ground up in 5 days. I wanted to build an app without using client side data and embedded data in MongoDB. I used pure jQuery DOM manipulation from the database to alter the UI.
